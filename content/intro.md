# Micro-Profiling

This tutorial was first designed for teaching the lab members of Dr. Hani El-Nezami from [University of Hong Kong](https://www.hkuelnlab.com/our-team),  and [University of Eastern Finland](https://uefconnect.uef.fi/en/hani.el-nezamy/). As one of the Phd students supervised by Hani, This git repo provided a tutorial about how to utilise the bioinformatics tools to understand a microogranism's profile through using the genomics and other omixs data 

This tutorial will mainly focus on using online webserver tools and R:

1. Safety assessment of the probiotics
2. Genome annotation and metabolic potentials
